# Calculadora JS - Práctica de Programación

Este es un proyecto de una calculadora funcional construida con **HTML5, CSS3 y JavaScript vanilla**. El objetivo principal de este repositorio es practicar la manipulación del DOM, el manejo de eventos y la implementación de lógica matemática en el frontend.

## Características
* Interfaz moderna y responsiva (Dark Mode).
* Operaciones básicas: suma, resta, multiplicación y división.
* Manejo de decimales y limpieza de pantalla (AC).
* Efectos visuales en botones para mejorar la experiencia de usuario (UX).

## Tecnologías utilizadas
* **HTML5**: Estructura semántica.
* **CSS3**: Flexbox, Grid y animaciones de transición.
* **JavaScript (ES6)**: Lógica de cálculo y manipulación del DOM.
